How to Play: Power Game

Setup: In groups of more than 3, take turns rolling two dice. The left die determines the base number, and the right die determines the exponent (index/power).

Game 1: Power Duel

Each player rolls twice. Multiply the outcomes and keep them in index form. The player with the highest base and index wins.

Game 2: Power Division

Each player rolls twice. Divide the first outcome by the second and simplify in index form. The player with the lowest base and index wins.

Game 3: Power Chain

Each player rolls twice. The first roll determines the base. The exponents stack: the base is raised to the second number, then raised again to the next roll's number, and finally raised again to the last number. The result should be simplified in index form. The player with the highest base and index wins.

Game 4: Negative Power Challenge

Each player rolls twice. In the first roll, raise the base to the negative of the exponent. In the second roll, do the same and then divide both results. The player with the lowest base and index wins.
